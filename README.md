# 🏠 RealEstate-AI-Retoucher

Professional-grade color correction for interior photography, powered by Google's Gemini Vision API.

## 🚀 Overview
RealEstate-AI-Retoucher is an AI-powered tool designed specifically for real estate professionals. Unlike standard AI filters that flatten images into a dull, gray aesthetic, this tool uses custom logic to surgically neutralize unwanted artificial color casts—such as tungsten lamp halos—while preserving the natural integrity of your interior materials and designer paint colors.

Delivers bright, crisp, high-end, MLS-ready results in seconds.

## 🌟 Key Features
- **Smart Neutralization:** Automatically removes "ugly" yellow/orange lamp spill while protecting the natural decorative glow of light sources.
- **Material Integrity:** Intelligent detection ensures that wood, stone, and designer wall colors remain vivid and true to life.
- **MLS-Ready Quality:** Optimizes highlight and shadow recovery to produce inviting, luxury-grade photography.
- **Automated Workflow:** Eliminates hours of manual post-processing, perfect for high-volume listing photographers.

## 🛠️ Technology Stack
- **Framework:** Streamlit (Python)
- **AI Model:** Google Gemini (Generative AI SDK)
- **Image Processing:** PIL (Python Imaging Library)

## 📋 Prerequisites
Before running the project, ensure you have:
1. Python 3.10+ installed.
2. A Google AI Studio API Key.
